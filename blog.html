The Struggles Nobody Talks About

1. Ollama Installation Took 3 Days

Termux, proot-distro, Ubuntu, 404 errors, wrong architecture (I downloaded amd64 instead of arm64) — I tried everything. On day 3, it finally worked. The moment I saw "Hello, how can I assist you?" on my phone, I felt like I had won.

2. Admin Panel Login Refused to Work

10 days. Firebase rules, managers collection, authentication — one small mistake and nothing worked. Finally fixed by creating a new admin account and updating Firestore security rules to allow read, write: if true; for testing.

3. Telegram Bot Deployment Failed 10 Times

Render, environment variables, secret files, start commands — a single typo in the start command crashed the entire bot. The error log showed "Exited with status 1" for hours before I realized the filename was wrong.

4. Getting Users is Harder than Coding

I built the product. It works. But nobody knows it exists. Now I'm learning marketing — DMs, content, videos, referrals. This is a whole new skill I'm still learning.

What I Learned (So Far)

  • 📌 Start where you are — Don't wait for perfect setup. Perfect is the enemy of done.
  • ⚡ Fail fast, fix faster — Every error teaches something. Read the logs. Try again.
  • 🌍 Build in public — Document your journey. People connect with real stories, not polished marketing.
  • 🤖 Automate everything — AI agents work while you sleep. Let them handle repetitive tasks.
  • 🔄 Rejection is redirection — Failed attempts are not failure. They're data for the next attempt.
